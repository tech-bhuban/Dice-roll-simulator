# Dice Roll Simulator

## What it does
Virtual dice roller with animation and history.

## Features
- Random dice roll (1-6)
- Animated roll effect
- Shows dice symbols (⚀-⚅)
- History of last 5 rolls

## How to use
- Click "Roll Dice" button
- See dice face and number
- Check last rolls history

## Dice Symbols
- ⚀ = 1
- ⚁ = 2
- ⚂ = 3
- ⚃ = 4
- ⚄ = 5
- ⚅ = 6
